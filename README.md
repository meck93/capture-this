## Requirements

- macOS 15+
- Xcode + Command Line Tools
- [mise](https://mise.jdx.dev) for tool management

## Getting started

```bash
mise install
mise run generate
```

## Common commands

```bash
mise run build
mise run test
mise run lint
mise run format
```

## Release artifacts (local)

```bash
mise run generate
mise run release-build
mise run package
```

Artifacts are written to `artifacts/`.
